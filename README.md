EyeCandy

Aplicación web desarrollada en ASP.NET Core MVC (C#) para administrar un catálogo de series: portadas, géneros, actores y directores, con carga de imágenes y persistencia en base de datos mediante Entity Framework Core.

¿Qué hace?


CRUD completo (crear, ver, editar, eliminar) para series, actores, directores y géneros.
Subida y almacenamiento de imágenes (portadas) asociadas a cada serie.
Relaciones entre entidades (una serie puede tener varios actores y géneros).
Migraciones de base de datos versionadas con Entity Framework.


Tecnologías


C# / .NET (ASP.NET Core MVC)
Entity Framework Core (migraciones, repositorios)
AutoMapper para mapeo entre modelos y entidades
SQL Server


Aprendizajes

Acá practiqué el patrón repositorio, el mapeo de modelos con AutoMapper, y el manejo de archivos (subida de imágenes) dentro de una aplicación con varias entidades relacionadas entre sí — una de las prácticas más completas de la carrera en cuanto a arquitectura de proyecto.
